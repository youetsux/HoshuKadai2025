# 砲台ゲーム - ゲーム数学補習課題

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📝 概要

この課題は、ゲームエンジニア1年生のゲーム数学補習課題です。
DxLibを使った砲台ゲームを作成しながら、以下の数学的概念を実践的に学習します。

### 学習目標

- 三角関数を使った角度から方向ベクトルへの変換
- ベクトルのスカラー倍、加算、減算
- 放物線運動（重力加速度）
- 2点間の距離計算
- 円同士の当たり判定

---

## 🎮 ゲーム内容

砲台から弾を発射して、的に当てるシンプルなゲームです。

### 操作方法
- **LEFT/RIGHT キー**: 砲台の角度を調整
- **UP/DOWN キー**: 弾の初速度を調整
- **SPACE キー**: 弾を発射

## 💻 環境構築

### 必要なもの

- Visual Studio 2022
- DXライブラリ (https://dxlib.xsrv.jp/)

### セットアップ手順

#### 1. プロジェクトの取得

**方法A: Visual Studio 2022 でクローン（推奨）**

1. Visual Studio 2022 を起動
2. 「リポジトリのクローン」をクリック
3. 「リポジトリの場所」に以下のURLを入力：
   ```
   https://github.com/youetsux/HoshuKadai2025.git
   ```
4. 「パス」で保存先フォルダを選択
5. 「クローン」ボタンをクリック

**方法B: ZIP ダウンロード**

1. [GitHubリポジトリ](https://github.com/youetsux/HoshuKadai2025)にアクセス
2. 「Code」→「Download ZIP」をクリック
3. ZIPを解凍

#### 2. DxLibのパスを設定

- プロジェクトのプロパティ → VC++ディレクトリ
- インクルードディレクトリとライブラリディレクトリにDxLibのパスを追加

#### 3. プロジェクトをビルド

- `HoshuKadai.sln` を開く
- ビルド → ソリューションのビルド

## 📖 課題の進め方

### ステップ1: ワークシートを読む

各ワークシートには、数式と実装手順が詳しく書かれています。
まずはワークシートをよく読んで、何を実装すべきか理解しましょう。

### ステップ2: コード内の TODO を探す

ソースコード内に `// TODO:` というコメントがあります。
そこが実装すべき箇所です。

### ステップ3: 実装する

ワークシートの説明を参考に、TOOの箇所を実装します。
わからない場合は、ヒントや数式を参考にしましょう。

### ステップ4: ビルドして実行

実装したらビルドして、実際に動作を確認します。
うまく動かない場合は、ワークシートのデバッグヒントを参考にしてください。
