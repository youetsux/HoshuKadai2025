# 砲台ゲーム - ゲーム数学補習課題

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ?? 概要

この課題は、専門学校ゲームエンジニア科1年生のゲーム数学補習課題です。
DxLibを使った砲台ゲームを作成しながら、以下の数学的概念を実践的に学習します。

### 学習目標

- ? 三角関数を使った角度から方向ベクトルへの変換
- ? ベクトルのスカラー倍、加算、減算
- ? 放物線運動（重力加速度）
- ? 2点間の距離計算
- ? 円同士の当たり判定

---

## ?? ゲーム内容

砲台から弾を発射して、的に当てるシンプルなゲームです。

### 操作方法
- **LEFT/RIGHT キー**: 砲台の角度を調整
- **UP/DOWN キー**: 弾の初速度を調整
- **SPACE キー**: 弾を発射

---

課題は4つのワークシートに分かれています：

1. [**ワークシート01: Cannon クラス（砲台）**](docs/WORKSHEET_01_Cannon_Problem.md)

3. [**ワークシート03: Target クラス（的）**](docs/WORKSHEET_03_Target_Problem.md)

4. [**ワークシート04: Stage クラス（当たり判定）**](docs/WORKSHEET_04_Stage_Problem.md)
   - 差ベクトル、2点間の距離、円の衝突判定

## ??? 環境構築

### 必要なもの

- Visual Studio 2019 以降
- DXライブラリ (https://dxlib.xsrv.jp/)

### セットアップ手順

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/youetsux/HoshuKadai2025.git
   ```

2. DxLibのパスを設定
   - プロジェクトのプロパティ → VC++ディレクトリ
   - インクルードディレクトリとライブラリディレクトリにDxLibのパスを追加

3. プロジェクトをビルド
   - `HoshuKadai.sln` を開く
   - ビルド → ソリューションのビルド

## ?? 課題の進め方

### ステップ1: ワークシートを読む

各ワークシートには、数式と実装手順が詳しく書かれています。
まずはワークシートをよく読んで、何を実装すべきか理解しましょう。

### ステップ2: コードに TODO を探す

ソースコード内に `// TODO:` というコメントがあります。
そこが実装すべき箇所です。

### ステップ3: 実装する

ワークシートの説明を参考に、TODOの箇所を実装します。
わからない場合は、ヒントや数式を見直しましょう。

### ステップ4: ビルド＆実行

実装したらビルドして、実際に動作を確認します。
うまく動かない場合は、ワークシートのデバッグヒントを参考にしてください。

## ? 完成チェックリスト

すべての実装が終わったら、以下を確認してください：

- [ ] コンパイルエラーがない
- [ ] 左右キーで砲台の角度が変わる
- [ ] 砲身が正しい方向を向いている
- [ ] スペースキーで弾が発射される
- [ ] 弾が放物線を描いて飛ぶ
- [ ] 弾が的に当たると色が変わる
- [ ] 弾が的に当たると「HIT!」が表示される
- [ ] 的に当たった後の弾は消える
- [ ] 画面外に出た弾は消える

## ?? 発展課題

基本課題が完成したら、以下に挑戦してみましょう：

1. **複数の的**
   - 異なる位置に3つの的を配置
   - それぞれ点数を変える

2. **スコア表示**
   - 何発で当てたか表示
   - 命中率の計算

3. **軌跡予測**
   - 発射前に弾道を点線で表示

4. **パーティクル効果**
   - 当たったときに爆発エフェクト

5. **サウンド**
   - 発射音、ヒット音、BGM
