# 砲台ゲーム - ゲーム数学補習課題

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

GE1Aゲーム数学低点学生向けの補習課題です。
DxLibを使った砲台ゲームを作成しながら、ベクトル演算と放物線運動を学びます。

---

## 🎮 デモ

砲台から弾を発射して、的に当てるシンプルなゲームです。

### 操作方法
- **LEFT/RIGHT キー**: 砲台の角度を調整
- **UP/DOWN キー**: 弾の初速度を調整
- **SPACE キー**: 弾を発射

## 🎯 学習目標

この課題を通して、以下の数学的概念を実践的に学習します：

### 三角関数
- 角度（ラジアン）から方向ベクトル $(\cos\theta, \sin\theta)$ への変換
- 度とラジアンの変換

### ベクトル演算
- ベクトルの加算: $\vec{a} + \vec{b}$
- ベクトルの減算: $\vec{a} - \vec{b}$
- スカラー倍: $k \cdot \vec{v}$
- ベクトルの長さ: $\|\vec{v}\| = \sqrt{v\_x^2 + v\_y^2}$

### 物理シミュレーション
- 等速運動: $\vec{p}_{\text{new}} = \vec{p}\_{\text{old}} + \vec{v}$
- 加速度運動: $\vec{v}\_{\text{new}} = \vec{v}\_{\text{old}} + \vec{a}$
- 重力による放物線運動

### 当たり判定
- 2点間の距離
- 円同士の衝突判定: $\text{distance} \leq r_1 + r_2$

## 📥 ダウンロード・セットアップ

### 1. プロジェクトをダウンロード

**方法A: ZIP ダウンロード**
1. [GitHubリポジトリ](https://github.com/youetsux/HoshuKadai2025)にアクセス
2. 「Code」→「Download ZIP」をクリック
3. ZIPを解凍

**方法B: Git クローン**
```bash
git clone https://github.com/youetsux/HoshuKadai2025.git
```

### 2. DxLib の設定

DxLibのパスをプロジェクトに設定してください。
- プロジェクトのプロパティ → VC++ディレクトリ
- インクルードディレクトリとライブラリディレクトリにDxLibのパスを追加

### 3. 初期状態の確認

`HoshuKadai.sln` を開いてビルドしてみてください。

⚠️ **重要:** 初期状態では **コンパイルエラーが発生します**。これは正常です！

```
エラー内容の例：
- C2059: 構文エラー: '{'
- C2065: 定義されていない識別子
- E0040: 識別子が必要です
```

**なぜエラーが出るのか？**
- ヘッダーファイル（.h）が未完成の状態だから
- これから皆さんが完成させていきます

## 📝 課題の進め方

### 🚨 重要: 2段階で進めます

この課題は **フェーズ1（.h）** と **フェーズ2（.cpp）** の2段階構成です。

---

### 【フェーズ1】ビルドできるようにする（目安: 30分）

**目標:** コンパイルエラーを解消して、プログラムが起動できる状態にする

**作業内容:**
1. `Cannon.h` の穴埋め（ワークシート01 問題1）
2. `Bullet.h` の穴埋め（ワークシート02 問題1）
3. `Target.h` の穴埋め（ワークシート03 問題1）

**作業手順:**
1. 各ワークシートの「問題1」を読む
2. 対応する.hファイルの`/* TODO: */`部分を埋める
3. すべての.hファイルが完成したらビルドする

**達成条件:**
- [ ] コンパイルエラーがなくなる
- [ ] プログラムが起動する
- [ ] ウィンドウが表示される（ただし砲台は動かない）

💡 **ヒント:** 3つの.hファイルは似た構造なので、1つ理解すれば残りもスムーズです！

---

### 【フェーズ2】機能を実装する（目安: 2〜3時間）

**目標:** ゲームを完成させる

**作業内容:**
1. ワークシート01: Cannon.cpp の実装（砲台を動かす）
2. ワークシート02: Bullet.cpp の実装（弾を飛ばす）
3. ワークシート03: Target.cpp の実装（的を配置）
4. ワークシート04: Stage.cpp の実装（当たり判定）

**作業手順:**
1. ワークシートを順番に読む
2. TODOコメントがある関数を実装する
3. 実装したらビルドして動作確認
4. 次のワークシートに進む

**達成条件（最終ゴール）:**
- [ ] 左右キーで砲台の角度が変わる
- [ ] 砲身が正しい方向を向いている
- [ ] スペースキーで弾が発射される
- [ ] 弾が放物線を描いて飛ぶ
- [ ] 弾が的に当たると色が変わる
- [ ] 弾が的に当たると「HIT!」が表示される

## 🚀 発展課題

基本課題が完成したら、以下に挑戦してみましょう：

1. **複数の的**
   - 異なる位置に3つの的を配置
   - それぞれ点数を変える

2. **スコア表示**
   - 何発で当てたか表示
   - 命中率の計算

3. **軌跡予測**
   - 発射前に弾道を点線で表示

4. **的を動かす**
   - 左右に往復運動
   - $\sin$ 波で上下運動

5. **パーティクル効果**
   - 当たったときに爆発エフェクト

6. **サウンド**
   - 発射音、ヒット音、BGM
