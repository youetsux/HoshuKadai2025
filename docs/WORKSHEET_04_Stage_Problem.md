## 📐 当たり判定の数学

### 円同士の衝突判定の原理

2つの円が衝突しているかを判定する条件：

$
\text{中心間の距離} \leq \text{半径の和}
$

### 計算手順

**ステップ1: 差ベクトルを求める**

弾の位置 $$\\vec{p_b}$$ と 的の位置 $$\\vec{p_t}$$ の差：

$
$\\vec{r}$ = $\\vec{p_b}$ - $\\vec{p_t}$
$

成分で書くと：
$
$\\vec{r}$ = ($p_{bx}$ - $p_{tx}$,\ $p_{by}$ - $p_{ty}$)
$

**ステップ2: 距離を求める**

差ベクトルの長さが2点間の距離：

$
d = |$\\vec{r}$| = $\\sqrt{$r_{x}$^2 + $r_{y}$^2}$
$

**ステップ3: 判定**

$
d \leq $r_{b}$ + $r_{t}$ \quad \Rightarrow \quad \text{衝突！}
$

- $$r_{b}$$: 弾の半径
- $$r_{t}$$: 的の半径

---

## 💡 具体例で理解する

### 例1: 衝突している場合

**条件：**
- 弾の位置: (100, 100)、半径: 5
- 的の位置: (108, 100)、半径: 10

**計算：**

1. 差ベクトル:
   $
   $\\vec{r}$ = (100, 100) - (108, 100) = (-8, 0)
   $

2. 距離:
   $
   d = $\\sqrt{(-8)^2 + 0^2}$ = $\\sqrt{64}$ = 8
   $

3. 半径の和:
   $
   $r_{b}$ + $r_{t}$ = 5 + 10 = 15
   $

4. 判定:
   $
   8 \leq 15 \quad \Rightarrow \quad \text{衝突！}
   $

### 例2: 衝突していない場合

**条件：**
- 弾の位置: (100, 100)、半径: 5
- 的の位置: (130, 100)、半径: 10

**計算：**

1. 差ベクトル:
   $
   $\\vec{r}$ = (100, 100) - (130, 100) = (-30, 0)
   $

2. 距離:
   $
   d = $\\sqrt{(-30)^2 + 0^2}$ = 30
   $

3. 半径の和:
   $
   $r_{b}$ + $r_{t}$ = 5 + 10 = 15
   $

4. 判定:
   $
   30 > 15 \quad \Rightarrow \quad \text{衝突していない}
   $

### 数式の確認

速度ベクトル $$\\vec{v}$$ は、方向ベクトル $$\\vec{d}$$ と速さ $s$ から：

$
$\\vec{v}$ = s \cdot $\\vec{d}$
$

```cpp
void Stage::ShootBullet(Cannon* cannon)
{
    if (cannon == nullptr) return;

    // 砲身の先端位置を取得
    Vector2D pos = cannon->【①】;
    
    // 方向ベクトルを取得（単位ベクトル）
    Vector2D dir = cannon->【②】;
    
    // 速度ベクトル = 方向ベクトル × 速さ
    Vector2D v = Math2D::【③】(dir, cannon->【④】);

    // 弾を生成してオブジェクトリストに追加
    Bullet* b = new Bullet(pos, v, BULLET_COLOR, BULLET_RADIUS);
    AddObject(b);
}
```

**ヒント：**
- ①: 砲身の先端位置を取得する関数（Cannonクラスのメソッド）
- ②: 方向ベクトルを取得する関数（Cannonクラスのメソッド）
- ③: ベクトルにスカラーを掛ける関数（Math2D の関数）
- ④: 弾の初速度を取得する関数（Cannonクラスのメソッド）

## 💥 問題2: Bullet_vs_Target() の実装（重要！）

弾と的の当たり判定を実装します。これが最も重要な問題です！

### 計算手順の復習

1. 差ベクトル: $$\\vec{r}$ = $\\vec{p_b}$ - $\\vec{p_t}$$
2. 距離: $d = |$\\vec{r}$|$
3. 判定: $d \leq $r_{b}$ + $r_{t}$$

```cpp
void Stage::Bullet_vs_Target()
{
    // 有効な弾と的を集めるリスト（この部分は提供済み）
    std::vector<Bullet*> activeBullets;
    std::vector<Target*> targets;

    // オブジェクトリストから弾と的を抽出
    for (auto& obj : objects)
    {
        if (obj->GetType() == OBJ_TYPE::BULLET)
        {
            Bullet* b = (Bullet*)obj;
            if (b->IsActive())
            {
                activeBullets.push_back(b);
            }
        }
        else if (obj->GetType() == OBJ_TYPE::TARGET)
        {
            Target* t = (Target*)obj;
            if (!t->IsHit())
            {
                targets.push_back(t);
            }
        }
    }

    // すべての弾と的の組み合わせで当たり判定
    for (auto& bullet : activeBullets)
    {
        for (auto& target : targets)
        {
            // ステップ1: 差ベクトルを計算
            Vector2D diff = Math2D::【⑤】(bullet->GetPos(), target->GetPos());
            
            // ステップ2: 距離を計算（ベクトルの長さ）
            float dist = Math2D::【⑥】(【⑦】);
            
            // ステップ3: 衝突判定距離（半径の和）
            float collisionDist = bullet->【⑧】 + target->【⑨】;
            
            // ステップ4: 衝突判定
            if (dist 【⑩】 collisionDist)
            {
                target->SetHit();
                bullet->SetInactive();
                isHit = true;
                gameScore_ += 100;
            }
        }
    }
}
```

**ヒント：**
- ⑤: 2つのベクトルを引く関数（Math2D の関数）
- ⑥: ベクトルの長さを求める関数（Math2D の関数）
- ⑦: ⑥の関数に渡すベクトル（ステップ1で計算した変数）
- ⑧: 弾の半径を取得する関数（Bulletクラスのメソッド）
- ⑨: 的の半径を取得する関数（Targetクラスのメソッド）
- ⑩: 距離と半径の和を比較する演算子（<, <=, >, >= のいずれか）

**重要な考察：**
- なぜ `dist < collisionDist` ではなく `dist <= collisionDist` でしょうか？
- 等号（=）を含めるべきでしょうか？
